# Bundled UI Fonts

DataMoat bundles these fonts for consistent local UI rendering across macOS,
Windows, and Linux. Each font keeps its upstream license.

## Onest

- Source: https://github.com/google/fonts/tree/main/ofl/onest
- Designer: Martin Sznaider, Indian Type Foundry
- License: SIL Open Font License 1.1
- File: Onest.ttf

## JetBrains Mono

- Source: https://github.com/JetBrains/JetBrainsMono
- License: SIL Open Font License 1.1
- File: JetBrainsMono-Regular.ttf
