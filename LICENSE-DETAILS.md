# DataMoat License Details

This page keeps the README short. It is a plain-English summary, not a replacement for the license.

DataMoat is source-available under **Business Source License 1.1 (`BUSL-1.1`)** with an **Additional Use Grant**.

The short version:

- Personal use is allowed.
- Internal company use is allowed.
- Uses outside those allowed uses require a separate commercial license from the licensor.

We chose **BUSL-1.1** to keep the code auditable while reducing the risk of misleading repackaged builds, malware clones, and unsupported commercial forks of a security-sensitive local archive tool. All application code is public for review.
